CONFIG DO GIT
-Instala o Git no pc
-Abre o Bash

$ git config --global user.name "YOUR NAME"
$ git config --global user.email "YOUR EMAIL ADDRESS"


GIT TUTORIAL

-Cria o repositorio no github
-Entra na pasta no pc que quer que o projeto esteja
-Clone com: git clone https://github.com/<username>/site.git


COMANDOS DO GIT

-git status (ver o status dos arquivos)
-git add . (o . add todos os arquivos)
-git commit -m "Mensagem" (faz o commit com a mensagem)
-git push (empurra tudo pro github)


CRIAR PROJETO NO VSCODE

-No VsCode acesse a pasta raiz do projeto clonado do git. Ex: projeto esta em Doc/ExemploApp, entao acesso a Doc/
-Crie o projeto com: react-native init ExemploApp (dê um 'y' pra sobreescrever a pasta)


APP Tutorial

-Cria Routes.js
-Edita a App.js
-Cria pasta src > components (ou outras)
-Cria a Initial.js em components
-Verifica quais dependencias e começa a programar


ADMOB TUTORIAL

-Criar o app na pagina do admob.
-Copiar o id do app admob e colocar no manifest do app dentro do <meta-data> (copiar do massagem)
-Criar a campanha no admob.
-Copiar o id da campanha e colocar no app, dentro da tag do <AdMobBanner> (Copiar do massagem)

GERAR O APK 
-Seguir algum tutorial de gerar apk, primeiro tem que gerar o keystore e SALVAR ELE PRA QUANDO PRECISAR ATUALIZAR.
